# Installation
Depending on your Operating System, you need to place the stuff inside the zip in a certain directory.
The root directory needs to look like this:

->mpv

-->input.conf

-->mpv.conf

--->Shaders

--->script-opts

--->scripts


# Windows Path
"C:\Users\YOURNAME\AppData\Roaming\mpv"

# Linux Path
/home/.config/mpv

# Mac Path
/USERNAME/.config/mpv
